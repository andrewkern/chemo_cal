Print Test Instructions

To test the print layout fixes:

  1. Open the main application at frontend/index.html
  2. Select a drug regimen from the dropdown
  3. Set your desired start date
  4. Click "Generate Schedule"
  5. Click "Print Calendar" or use Ctrl/Cmd + P
  6. In the print preview, check that:
    • Day labels (Sun, Mon, Tue, etc.) align properly with the calendar cells below them
    • All 7 days of the week are visible without being cut off
    • The calendar uses landscape orientation
    • Events are visible with appropriate colors

The CSS fixes applied:

  • Fixed column header cell alignment with center text alignment
  • Set consistent padding and vertical alignment for headers
  • Applied fixed table layout to ensure consistent column widths
  • Set explicit 14.2857% width (100% / 7 days) for both headers and day cells
  • Removed any CSS transforms that could cause misalignment
  • Added proper borders to all table cells
